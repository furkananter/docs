description: 'A customizable Rainbow Button component that provides haptic feedback and opacity effects on press.'
---

`RainbowButton2` is a customizable button component built for React Native applications. It provides haptic feedback on press and visual feedback through opacity and scaling effects.

## Installation

Ensure you have the necessary dependencies:

```bash
npm install expo-haptics
```

## Import

`import PressableOpacity from '@/components/common/buttons/pressable-opacity`
