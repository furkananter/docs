# Sign in with Google on Expo React Native IOS

First make sure you have completed the [Google OAuth Web](/auth/oauth/google/web) page.
- We need Supabase to be setup.

<Steps>
    <Step>
        First of all, you need to create a new project in the Google Cloud Console if you don't have one already.
        - We created a new project in [RevenueCat Integration Android](/monetization/in-app-purchases/android) page.
        If you want you can follow this first.
        <Frame>
        <img src="/images/setup/googleoauth/ios/step1.png" alt="Step 1" className="w-full h-full rounded-xl" />
        </Frame>
    </Step>
    <Step>
        Click the `APIs & Services` menu.
        <Frame>
            <img src="/images/setup/googleoauth/ios/step2.png" alt="Step 2" className="w-full h-full rounded-xl" />
        </Frame>
    </Step>
    <Step>
        Click the `Credentials` menu.
        <Frame>
            <img src="/images/setup/googleoauth/ios/step3.png" alt="Step 3" className="w-full h-full rounded-xl" />
        </Frame>
    </Step>
    <Step>
        Then click the `Create Credentials` button.
        - Please select `OAuth client ID`
        <Frame>
            <img src="/images/setup/googleoauth/ios/step4.png" alt="Step 4" className="w-full h-full rounded-xl" />
        </Frame>
    </Step>
    <Step>
        But you need to `Configure Consent Screen` first.
        <Frame>
            <img src="/images/setup/googleoauth/ios/step5.png" alt="Step 5" className="w-full h-full rounded-xl" />
        </Frame>
    </Step>
    <Step>
        Create IOS oauth client id.
     - Write a name of your app.
     - Write your `bundle id` in the `Bundle ID` field.
     - Click the `Create` button.
        <Frame>
            <img src="/images/setup/googleoauth/ios/step6.png" alt="Step 6" className="w-full h-full rounded-xl" />
        </Frame>
    </Step>
    <Step>
    Copy Client ID and Paste it to the `.env` file.
        <Frame>
            <img src="/images/setup/googleoauth/ios/step7.png" alt="Step 7" className="w-full h-full rounded-xl" />
        </Frame>
    ```{1}
        EXPO_PUBLIC_GOOGLE_OAUTH_IOS_CLIENT_ID=some-id.apps.googleusercontent.com
    ```
    </Step>
    <Step>
        Then you need to select `IOS oAuth Client` first. 
        <Frame>
            <img src="/images/setup/googleoauth/ios/step8.png" alt="Step 8" className="w-full h-full rounded-xl" />
        </Frame>
    </Step>
    <Step>
        <Frame>
            <img src="/images/setup/googleoauth/ios/step9.png" alt="Step 9" className="w-full h-full rounded-xl" />
        </Frame>
    Copy `IOS URL Scheme` and go to `app.json` file.
    - Add here:
    ```{4}
       [
        "@react-native-google-signin/google-signin",
        {
          "iosUrlScheme": "com.googleusercontent.apps.some-id" // Here
        }
      ]
    ```
    <Check>
        You have successfully set up Google OAuth for your Expo React Native IOS Client. ðŸŽ‰
    </Check>
    <Tip>
        Go and setup Android OAuth.
    </Tip>
    </Step>
</Steps>
