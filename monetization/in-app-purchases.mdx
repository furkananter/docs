# RevenueCat

We are using [RevenueCat](https://www.revenuecat.com/) to handle in-app purchases.

<Steps>
    <Step>
        First, go to [RevenueCat](https://www.revenuecat.com/) and create a account.
    </Step>
    <Step>
        After you create your account, create a new project.
    </Step>
    <Step>
        After creating your project, you need to create a new app for iOS and Android.
    </Step>
    <Step>
        Then, you need to configure your apps.
    </Step>
    <Step>
        <Warning>
            I can give you steps but original RevenueCat docs are better.
        </Warning>
        <a href="https://docs.revenuecat.com/docs/ios-quickstart">RevenueCat Docs</a>
    </Step>
</Steps>
