# Chat App Template

Create engaging messaging applications with our production-ready chat template.

## Features

- 💬 Real-time messaging
- 👥 Group chats
- 📸 Media sharing
- 📍 Location sharing
- 🔔 Push notifications
- 👀 Read receipts
- 🔍 Message search
- 📱 Offline support

## Architecture

The chat template uses:
- Supabase Real-time for messages
- Expo FileSystem for media caching
- Push notifications for message alerts
- SQLite for offline message storage
